# API

## Get all elements by tag type


`Scraper(url).find_tags_by_name(tag_name)`


Example output:

```
[
    <h3 class="country-name">
        <i class="flag-icon flag-icon-ad"></i>
            Andorra
    </h3>, 
    <h3 class="country-name">
        <i class="flag-icon flag-icon-ae"></i>
            United Arab Emirates
    </h3>
]
```
